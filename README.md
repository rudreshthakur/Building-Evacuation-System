Building Evacuation System

A smart building evacuation system developed using Python, NetworkX, and Dijkstra Algorithm.

This project represents a building floor as a graph where:
Nodes represent rooms, corridors, staircases, and exits
 Edges represent paths/connections between locations

The system finds the safest and shortest evacuation route while avoiding blocked or fire-affected areas.

 Features
 Graph-based building layout
 Fire/blocked node avoidance
 Shortest safe path calculation
 Dijkstra Algorithm implementation
 Distance calculation
 Graph visualization using Matplotlib

 Technologies Used
 Python
 NetworkX
 Matplotlib

 Example
Current Location: Room101  
Blocked Location: CorridorB

Safe Route:
Room101 -> Hallway -> Staircase -> ExitA

Total Distance: 8 units

Applications
 Smart Buildings- Emergency Evacuation Systems
 Indoor Navigation
 Fire Safety Systems
